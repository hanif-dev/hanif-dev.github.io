Who Am I

I'm a Full-Stack Software Engineer and Cyber Threat Intelligence (CTI) Analyst building systems that are technically rigorous and socially meaningful. My engineering work spans React/Next.js frontends, Java Spring Boot and Laravel backends, cloud deployments on AWS and GCP, and compliance-grade system design.
